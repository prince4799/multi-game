# Tetris Action Button Assets

Drop your PNG files here. The shell will automatically use them
once they are placed in this folder.

Expected files:
  rotate-cw.png   → ROTATE button  (games-config key: ArrowUp)
  drop.png        → DROP button    (games-config key: Space)
  rotate-ccw.png  → ROT ↺ button  (games-config key: z)

Path from gamerzArena root:
  games/puzzle/tetris/assets/rotate-cw.png
  games/puzzle/tetris/assets/drop.png
  games/puzzle/tetris/assets/rotate-ccw.png

If a PNG is missing the button automatically falls back to the
unicode icon (↻ / ⬇ / ↺) so nothing breaks while you're still
adding assets.
